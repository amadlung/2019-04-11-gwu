2. Data Carpentry Genomics Workshop at George Washington University¶

2.1. 2019-04-11-gwu¶

This website contains materials for the Data Carpentry Genomics Workshop at George Washington University two-day course (9:00 am-5:00pm), to be held held 11-12th April, 2019.

We are very excited to have you here and hope that we can get through a lot of material by the end of the workshop.

This workshop runs under a Code of Conduct. Please respect it and be excellent to each other!

Twitter hash tag: #DCGWU2019
